# stashmd

A curated directory of AI skills where every skill proves itself with before/after demos.

## The Idea

Skills are instructions that make AI agents better at specific tasks. But how do you know if a skill actually works?

**stashmd shows you.** Every skill has a live demo that demonstrates its transformation.

## Featured Skills

- **frontend-design** - Stop making sites that look like every other AI generation
- **code-simplifier** - Refine code for clarity without changing behavior
- **remotion** - Best practices for video creation in React

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

```bash
npm run build
vercel deploy
```
